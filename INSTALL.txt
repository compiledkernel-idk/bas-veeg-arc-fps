========================================
HOSTING A PRIVATE SERVER (CLASSMATES ONLY)
========================================

FOR THE HOST:
1. Double-click "START-CLASSMATES-SERVER.bat" (Windows)
   Or "START-CLASSMATES-SERVER.sh" (Linux)

2. Server starts on port 26000
   - Max 13 players (one for each classmate!)
   - Private (not listed publicly)
   - Allows ping up to 500ms

3. Share your IP address:
   Windows: Open Command Prompt, type: ipconfig
   Linux: The script shows your IPs automatically

4. Tell your classmates your IP (e.g., 192.168.1.100)

FOR FRIENDS JOINING:
1. Launch game: "PLAY-BAS-VEEG-ARC.bat"
2. Go to: Multiplayer → Server List
3. Click "Search LAN" (if on same WiFi)
   OR
   Click "Address" → Enter host's IP → Join
4. Select your character
5. PLAY!

========================================
TROUBLESHOOTING
========================================

Q: Characters don't appear?
A: Make sure BasVeegArc-v0.2-BRANDED.pk3 is in the data/ folder

Q: Can't join friend's server?
A: Check you're on same WiFi network
   Check Windows Firewall allows Xonotic
   Try manual IP entry

Q: Game won't start?
A: Try xonotic-wgl.exe instead
   Update graphics drivers
   Run as Administrator

Q: All characters look the same?
A: Normal! They use same model but have different names/stats
