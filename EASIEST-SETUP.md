# EASIEST SETUP - 2 STEPS!

## For Your Classmates (Super Easy!)

### STEP 1: Download
1. Go to: https://github.com/compiledkernel-idk/bas-veeg-arc-fps/releases
2. Download **`BasVeegArc-v0.2-COMPLETE.zip`** (or will be available soon)
3. Extract the ZIP file to anywhere (e.g., Desktop or C:\Games\)

### STEP 2: Play!
**Double-click:** `PLAY-BAS-VEEG-ARC.bat` (Windows) or `PLAY-BAS-VEEG-ARC.sh` (Linux)

**That's it! Game launches automatically!**

---

## To Host Server for Friends

**Double-click:** `START-CLASSMATES-SERVER.bat` (Windows) or `START-CLASSMATES-SERVER.sh` (Linux)

Share your IP address with friends (shown on screen)

---

## To Join Friend's Server

1. Launch game (double-click PLAY script)
2. Click **Multiplayer** → **Server List** → **Search LAN**
3. Or enter friend's IP manually
4. Select your character
5. Play!

---
